# SLIME Roadmap

SLIME stands for **Systems Language for Interactive Multiplatform Execution**.

This roadmap outlines the planned evolution of the language and toolchain.

---

## v0.1.0 — Core Compiler Foundations
- [x] Grammar specification
- [x] Lexer
- [x] Parser
- [x] AST module
- [x] Type checker
- [x] Error handling
- [x] CLI tool
- [x] Documentation
- [x] WebAssembly text backend
- [x] WebAssembly binary backend

---

## v0.2.0 — Usability & Standard Library
- [ ] Expand standard library
- [ ] Improve example programs
- [ ] Add `run` command
- [ ] Improve diagnostics and error formatting

---

## v0.3.0 — Native Execution
- [ ] Native backend scaffold
- [ ] C backend output
- [ ] Native executable generation

- [ ] Playground
